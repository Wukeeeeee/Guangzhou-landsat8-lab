# 广州 Landsat 8 监督分类工作流

本文件是项目流程的简要索引。更详细的过程、截图和结果分析见 [classification_process.md](classification_process.md)。

## 1. 数据准备

- 打开两景 Landsat 8 影像并检查波段信息。
- 在 ENVI Data Manager 中确认多光谱波段、全色波段、空间参考和分辨率。
- 使用广州市行政边界作为后续裁剪范围。

## 2. 影像预处理

主要预处理步骤如下：

1. 辐射定标：将 DN 值转换为辐亮度数据。
2. FLAASH 大气校正：得到更适合分类使用的反射率影像。
3. 全色融合：利用全色波段提升空间细节。
4. 两景影像镶嵌：形成连续覆盖广州市的影像。
5. 研究区裁剪：按广州市边界提取研究区。

## 3. ROI 样本选取

本次实验设置五类地物：

- 水体
- 植被
- 耕地
- 城镇用地
- 裸地

样本选取时尽量保证空间分布分散，但从最终结果看，复杂场景样本覆盖仍不足，尤其是江岸、库岸、阴影水体、浅水、裸地、道路和城郊过渡带。

## 4. 监督分类方法

本实验完成三种监督分类：

| 方法 | 说明 |
| --- | --- |
| 最大似然法 | 基于训练样本统计特征进行分类 |
| SVM | 使用 RBF 核函数处理非线性类别边界 |
| 神经网络 | 通过迭代训练拟合类别边界 |

## 5. 精度评价

| 方法 | 总体精度 OA | Kappa 系数 |
| --- | ---: | ---: |
| 最大似然法 | 94.9920% | 0.9348 |
| SVM | 96.5889% | 0.9556 |
| 神经网络 | 98.5637% | 0.9814 |

神经网络在验证样本上精度最高，但局部典型区域仍存在误分。因此本项目同时保留精度评价和目视误差分析。

## 6. 噪声处理

对三种分类结果使用 Majority/Minority Analysis 进行椒盐噪声处理。后处理可以减少孤立像元，使图斑更连续，但不能修正训练样本不足导致的根本误分。

## 7. 结果评价

本次结果适合作为监督分类流程和误差分析示例，不宜直接作为高精度土地覆盖制图成果。

- 最大似然法：图斑最碎，混分较明显。
- SVM：总体精度居中，但局部水体轮廓保持相对较好。
- 神经网络：验证样本精度最高，但江岸、水库和谷地等局部区域仍存在明显误分。
