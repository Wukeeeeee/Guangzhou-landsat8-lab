# 广州 Landsat 8 国土类型遥感监督分类实验

这是一个 GIS / 遥感课程实训项目，使用 ENVI 对广州市 Landsat 8 影像进行预处理、ROI 样本选取、监督分类、精度评价和结果误差分析。

本项目更偏向学习归档和方法对比。分类结果用于记录完整实验流程，不作为正式生产级土地覆盖制图成果。

## 项目内容

本实验完成了以下工作：

- Landsat 8 影像读取、波段检查和元数据查看
- 辐射定标、FLAASH 大气校正、全色融合和两景影像镶嵌
- 基于广州市边界进行研究区裁剪
- 选取水体、植被、耕地、城镇用地、裸地五类 ROI 样本
- 使用最大似然法、SVM 和神经网络进行监督分类
- 对三种方法进行总体精度、Kappa、图斑噪声和典型误分区域对比
- 使用 Majority/Minority Analysis 进行椒盐噪声处理对比

## 典型误分区域

以下四宫格图用于展示真实局部问题。图像顺序均为：左上最大似然法，左下 SVM，右上神经网络，右下原始影像。

### 江岸城市混合区

![江岸城市混合区误分对比](ScreenShot/typical_misclassification_city_river.png)

该区域包含河流、桥梁、道路、建筑、绿地、裸露地表和岸线浅水区。三种方法均能识别主体水体的大致形态，但江岸两侧存在明显混分。

- 最大似然法：图斑较碎，城镇用地、耕地和裸地混分明显。
- SVM：水体轮廓相对连续，但岸线附近仍有零散误分。
- 神经网络：整体图斑较平滑，但在复杂江岸区域也不能完全避免不合理分类。

比较明显的问题是，江边部分区域被分成耕地或城镇用地，这与原始影像中的实际地物并不完全一致。该现象说明训练样本对江岸浅水、桥梁、岸线建筑、道路和绿地等复杂混合场景覆盖不足。

### 山地水库区

![山地水库区误分对比](ScreenShot/typical_misclassification_reservoir_mountain.png)

该区域中，SVM 对主水体轮廓保持相对较好；最大似然法图斑较碎；神经网络虽然总体精度最高，但局部水库和谷地区域仍存在不合理误分。

- 最大似然法：黄色和红色斑块较碎，容易把山地边缘、道路或裸露地表分成城镇用地或耕地。
- SVM：主水体范围最接近原始影像，但库岸、阴影和浅水区仍存在混分。
- 神经网络：验证样本精度最高，但本局部图中部分水库或谷地区域被误判，说明总体精度不能代表所有局部区域。

该图也暴露了一个比较严重的问题：部分河流/水库水体没有被稳定识别出来，尤其在库岸、山体阴影和谷地开阔区域，水体会被误分为耕地或其他类别。这说明水体样本可能偏向开阔、典型水面，没有充分覆盖深水、浅水、阴影水体和浑浊水等情况。

## 实事求是的结果评价

本次实验完成了完整的监督分类流程，结果可以用于方法对比和误差分析，但不适合作为高精度土地覆盖制图成果。

- 最大似然法：图斑破碎，椒盐噪声明显，城镇用地、耕地和裸地混分较多。
- SVM：总体精度居中，但局部视觉效果相对稳定，尤其在水体轮廓保持方面表现较好。
- 神经网络：验证样本下 OA 和 Kappa 最高，但在江岸、水库和谷地等复杂局部区域仍出现明显误分。

主要问题来自广州复杂地物、Landsat 8 空间分辨率限制、混合像元、样本代表性不足以及验证样本覆盖不全面。

## 后续改进方向

- 补充阴影水体、浅水、浑浊水、库岸、道路、裸地、工地和城郊过渡带 ROI 样本
- 让验证样本覆盖江岸、库岸和复杂城市混合区域
- 引入 NDWI / MNDWI 等水体指数
- 对光谱高度相近的类别进行合并或重新定义
- 分类后结合原始影像进行人工检查和必要修正
